### Claude Code uses Bun written in Rust now
⭐ 无 | 评分：9.9 | 📅 2026-07-19
Anthropic 的 Claude Code 已将运行时切换为用 Rust 编写的 Bun，实现技术栈的重大重构。
> 技术亮点：Bun 是一个用 Zig 编写、底层依赖 Rust 的 JavaScript 运行时，以启动速度和性能著称。此次迁移意味着 Claude Code 放弃了 Node.js 生态，转向更现代、更高效的运行时。Rust 的内存安全特性和 Bun 的快速启动能力，将显著提升 Claude Code 在复杂编码任务中的响应速度和稳定性。
